how to play

  1. click, press space, or press the up key to make norman jump
  2. avoid all the pipes and get as far as you can
  3. enjoy!

about

this game is an adaptation of an earlier game, flapper burger, made by the same programmer. the game is still under development and will have major changes over time. below are some planned updates:

  • music/sfx!
  • different areas w/ different pipe behaviors
  • account system to save progress
  • currency earned by getting farther which can be redeemed for different rat(ish) skins
  • boss battles!

floppy rat is still under development at https://github.com/AvynEbersold/floppy-rat. any contributions would be welcome!
